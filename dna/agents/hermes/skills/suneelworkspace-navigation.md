**SuneelWorkSpace Navigation and Operations**
=============================================

### When to use this skill

Use the SuneelWorkSpace Navigation and Operations skill when you need to navigate and interact with the 12-organ Human Body Architecture of SuneelWorkSpace.

### Step-by-step procedure

1. **Accessing the Workspace**:
	* Open a terminal or command prompt and type `ws 'natural language'` to access the workspace.
2. **Navigating the Organs**:
	* Use the `memory-search` command to search for specific organs or functions within the body architecture.
	* Use the `brain-inject` command to inject new information into the brain organ.
3. **Running Operations**:
	* Use the `autolab-run` command to run automated laboratory operations on specific organs.
	* Use the `evolution-start` command to initiate evolutionary processes within the body architecture.
4. **Monitoring Status**:
	* Use the `nerve-status` command to monitor the status of nerve connections and signals.
	* Use the `model-status` command to check the status of models and simulations running within the body architecture.

### Key commands and file paths

| Command | Description |

### Common issues and solutions

* **Issue:** Unable to access the workspace.
	+ Solution: Check that you have correctly typed the `ws 'natural language'` command and that your terminal or command prompt is set up to interact with SuneelWorkSpace.
* **Issue:** Memory search query returns no results.
	+ Solution: Ensure that your query is specific and accurate, and try rephrasing it if necessary.

### Success criteria

* The user can successfully access the workspace using natural language interface.
* The user can navigate and interact with the 12-organ Human Body Architecture of SuneelWorkSpace.
* The user can run automated laboratory operations on specific organs and initiate evolutionary processes within the body architecture.
* The user can monitor the status of nerve connections, signals, models, and simulations running within the body architecture.
